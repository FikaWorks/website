## Requirements

- [Hugo](https://gohugo.io)
- [Nodejs](https://nodejs.org)

## Getting started

```bash
# install dev dependencies (tailwind, postcss, etc.)
$ npm install

# run hugo
$ hugo serve
```

You can now access the website locally at
[http://localhost:1313](http://localhost:1313).
